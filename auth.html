STRAVA // AUTH

// One-time OAuth setup · BIO-ATHLETE 53

// 01 · Set Client ID

Paste your Strava Application Client ID (from strava.com/settings/api).

// 02 · Strava Authorization

You'll be redirected to Strava to grant access. After approving, you'll come back here automatically.

// 03 · Save Refresh Token

Copy the refresh_token below and save it as STRAVA_REFRESH_TOKEN in Netlify env vars.

Path: Netlify → Site Settings → Build & Deploy → Environment
